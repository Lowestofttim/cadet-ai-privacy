# Cadet AI — Privacy Policy (hosting)

This public repository hosts **only** the privacy policy for the
[Cadet AI](https://github.com/Lowestofttim/Cadet-AI) app, served via GitHub Pages.

- Live URL: **https://lowestofttim.github.io/cadet-ai-privacy/**
- Source page: [`index.html`](index.html)

The app's source code lives in a separate private repository. This repo is
public solely so the privacy policy has a stable, publicly reachable URL for the
Google Play **Data safety** form and for in-app links.

The canonical text is kept in sync with `docs/PRIVACY_POLICY.md` in the app repo
and the in-app Privacy Policy screen. When you edit the policy, update all three.
